# Database
Just a little database in C++ to figure out how they work and how to make them faster.

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/LittleSilver33/Database.git
    cd Database
    ```

2. **Install Dependencies**
    ```bash
    sudo apt-get update
    sudo apt-get install -y cmake g++ make
    ```

3. **Build the project:**
    ```bash
    mkdir build
    cd build
    cmake ..
    make -j$(nproc)
    ```

4. **Run tests:**
    ```bash
    ./all_tests
    ```
